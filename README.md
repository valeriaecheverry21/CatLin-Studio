# CatLin Studio - Estudio de Belleza y Bienestar

Sitio web institucional desarrollado con HTML5, SASS y Bootstrap 5.

## Servicios
- Manicura profesional
- Diseño de cejas
- Extensiones de pestañas
- Masajes relajantes

## Estructura del Proyecto
```
CatLin-Studio/
├── index.html
├── pagina1.html (Servicios)
├── pagina2.html (Nosotros)
├── pagina3.html (Contacto)
├── pagina4.html (Galería)
├── assets/
│   ├── css/styles.css (generado desde SASS)
│   ├── js/main.js
│   └── images/
├── scss/
│   ├── abstracts/ (_variables.scss, _mixins.scss, _placeholders.scss)
│   ├── base/ (_reset.scss, _typography.scss)
│   ├── components/ (_nav.scss, _card.scss, _button.scss)
│   ├── layout/ (_header.scss, _footer.scss, _grid.scss)
│   ├── pages/ (_home.scss, _services.scss, _about.scss, _contact.scss)
│   └── main.scss
└── package.json
```

## Tecnologías Utilizadas
- HTML5 semántico
- SASS (con variables, mixins, nesting, extend)
- Bootstrap 5.3 (personalizado)
- AOS (Animate On Scroll)
- Font Awesome 6
- Google Fonts (Poppins, Playfair Display)

## Instalación y Desarrollo

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Compilar SASS: `npm run build-css`
4. Modo desarrollo (watch): `npm run sass`
